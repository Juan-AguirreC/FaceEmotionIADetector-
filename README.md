## Dataset utilizado

El modelo fue entrenado utilizando el dataset **FER2013 (Facial Expression Recognition 2013)** disponible en la plataforma Kaggle.

Fuente:
https://www.kaggle.com/datasets/msambare/fer2013

Este dataset contiene imágenes de rostros clasificadas en siete categorías de emociones:

| Clase | Emoción |
|------|---------|
| 0 | Angry |
| 1 | Happy |
| 2 | Neutral |
| 3 | Sad |
| 4 | Surprise |

El conjunto de datos fue utilizado para el entrenamiento, validación y evaluación del modelo de reconocimiento automático de emociones faciales.
